# CaliperForge

**Precision contributions, every chain.**

CaliperForge is a sole-operator, AI-augmented contribution agency for
crypto protocols. We ship smart contracts, developer tooling, and
ecosystem PRs to the protocols building the open Web. AI involvement
is disclosed at point of use; full policy at
[caliperforge.com/ai-disclosure](https://caliperforge.com/ai-disclosure)
and in [./AI_DISCLOSURE.md](./AI_DISCLOSURE.md).

## What we ship now

- **`cf-invariants`** — AI-suggested stateful invariant testing for
  Cairo 2.x on top of snforge. 3 reference contracts deployed and
  Voyager-verified on Starknet Sepolia (2026-05-31). Apache-2.0.
  See [caliperforge/cf-invariants](https://github.com/caliperforge/cf-invariants).

## What we work on

- **Starknet** — Cairo 2.x tooling, snforge ecosystem PRs, security
  contributions. Second active ecosystem (locked 2026-05-31).
- **Solana** — Anchor programs, SPL tooling, Foundation-funded work.
- **Base / Optimism Superchain** — smart contracts, dev tooling,
  RetroPGF-eligible builds.

Active language coverage: Cairo, Rust/Anchor, Solidity, TypeScript.

## How we work

We use AI coding assistants as part of how we build. Specifications
and implementations are drafted with substantial AI assistance,
reviewed by Michael as operator-of-record, tested in cold-environment
reproductions, and submitted under Michael's accountability for the
code, the tests, and any follow-up.

AI involvement is labeled at the source — in commit messages, PR
descriptions, grant applications, and where applicable in the code
itself. Full policy: [./AI_DISCLOSURE.md](./AI_DISCLOSURE.md).
